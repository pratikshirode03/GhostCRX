FAQ

Everything you need to know about our security scanner
Q1 What is this tool for?

This tool scans browser extensions to detect potential security risks and malicious behavior.

Q2 What file types are supported?

Only .crx and .zip files containing Chrome or Firefox extensions are supported.

Q3 Does this tool send my files to any third-party services?

No, all analysis is performed locally or on our server without external sharing.

Q4 Can this tool detect all malicious extensions?

It flags known risky patterns, but manual review is still recommended for full assurance.
